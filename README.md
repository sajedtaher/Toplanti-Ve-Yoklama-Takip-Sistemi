# 📋 Meeting & Attendance Management System

## 📖 Project Overview

The **Meeting & Attendance Management System** is a web-based application developed using **CodeIgniter 4**, **PHP**, and **MySQL**. It was created during my Computer Engineering internship to simplify meeting organization, participant management, attendance tracking, agenda planning, and reporting processes for academic units.

The system enables authorized users to create meetings, manage participants, record attendance, organize agenda items, and generate attendance reports through an intuitive web interface.

---

## ✨ Features

- User authentication and role-based authorization
- Meeting creation and management
- Participant management
- Attendance tracking
- Agenda item management
- Decision recording
- Attendance reports
- Administrative management panel
- Responsive user interface
